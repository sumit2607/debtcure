# debtcureIn - Premium Legal Services Website

A high-trust, professional single-page website designed for an Indian legal consultation platform helping people facing loan harassment.

## Features
- **Premium Design**: Deep navy blue and emerald green trustworthy color palette.
- **Responsive**: Fully responsive mobile-first design.
- **Conversion Optimized**:
  - Sticky Header with CTA
  - Floating WhatsApp Button
  - Exit Intent Lead Capture Modal
  - Trust Badges
  - Social Proof/Testimonials
- **Fast Performance**: Pure HTML/CSS/JS with no heavy frameworks.
- **SEO Ready**: Semantic HTML, meta tags, and Schema.org markup.
- **Privacy Focused**: No external trackers by default.

## Project Structure
- `index.html`: Main landing page with all sections.
- `css/style.css`: Custom styling with CSS variables for easy theming.
- `js/app.js`: Interactivity, scroll animations, and modal logic.
- `assets/`: Directory for images.

## Customization
To customize the site:
1.  **Images**: Add your own images to `assets/` and update the `src` attributes in `index.html`. The hero section currently uses a high-performance SVG illustration.
2.  **Contact Info**: vital to update the phone numbers in `href="tel:..."` and WhatsApp links in `index.html`.
3.  **Form Handling**: The form currently alerts a success message. You will need to connect the `js/app.js` form submission logic to your backend API or a service like Formspree.

## How to Run
Simply open `index.html` in any modern web browser.
